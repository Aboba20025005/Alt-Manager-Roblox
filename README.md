# Roblox Alt Manager

Roblox Alt Manager - это вспомогательный инструмент для удобного запуска нескольких аккаунтов Roblox через отдельные браузерные профили.

Программа НЕ обходит системы Roblox и использует официальный и легальный способ работы через браузер.

--- Особенности ---

- Отдельный браузерный профиль для каждого аккаунта
- Удобный список альт-аккаунтов с быстрым переключением
- Вход через браузер (Chrome / Yandex)
- Запуск игры через Roblox Player
- Сохранение списка аккаунтов между запусками программы
- Поддержка языков: English, Русский, Українська
- Современный тёмный интерфейс
- Встроенное предупреждение об ответственности

--- Принцип работы ---

1. Для каждого аккаунта создаётся отдельный профиль браузера
2. Вы один раз входите в Roblox через браузер в этом профиле
3. Нажимаете кнопку Play в программе
4. Roblox Player запускается уже под нужным аккаунтом

Это официальный и рабочий способ, который используют многие игроки.

## FAQ
# Сохраняются ли логин и пароль в программе?

Нет, программа не хранит ваши пароли. Данные для входа сохраняются самим браузером в изолированном профиле.



# Можно ли запускать много аккаунтов одновременно?

Да, каждый аккаунт использует свой отдельный профиль браузера, что позволяет запускать несколько копий Roblox Player.



# Это безопасно для аккаунта?

Программа не взламывает Roblox, не использует читы и не вмешивается в клиент игры. Способ запуска является легальным.



# Поддерживаются ли другие браузеры?

В текущей версии поддерживаются только Google Chrome и Yandex Browser.
